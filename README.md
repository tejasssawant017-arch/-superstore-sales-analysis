# Superstore Sales Analysis

## Overview
End-to-end sales data analysis on 10,000+ retail orders using MySQL and Power BI.

## Tools Used
- Python (pandas, mysql-connector)
- MySQL
- Power BI

## What I Did
- Imported 10,000+ rows of retail sales data into MySQL using Python
- Wrote 10 SQL business queries to extract insights
- Built an interactive Power BI dashboard with 6 visualizations

## Key Insights
- West region generates highest sales
- Technology is the most profitable category
- Standard Class is the most used shipping mode
- Consumer segment contributes highest revenue

## Files
- import_data.py — Python script to import CSV into MySQL
- queries.sql — All 10 SQL analysis queries
- Dashboard screenshot — Power BI dashboard

## Skills Demonstrated
SQL, Python, Power BI, Data Analysis, Business Intelligence
